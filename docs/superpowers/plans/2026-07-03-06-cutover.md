## Task 1: Old-URL 301 redirects on the author site

- [ ] **Step 1: Add `redirects()` to `zacharymays-site/next.config.js`**

```javascript
const OCI = 'https://organizationalcoercionindex.org';
const nextConfig = {
  async headers() { /* existing */ return [{ source: '/:path*', headers: securityHeaders }]; },
  async redirects() {
    const paths = ['/oci', '/explore', '/org', '/research', '/research-system', '/compass', '/findings'];
    const rules = [];
    for (const p of paths) {
      rules.push({ source: p, destination: `${OCI}${p}`, permanent: true });
      rules.push({ source: `${p}/:path*`, destination: `${OCI}${p}/:path*`, permanent: true });
    }
    return rules;
  },
};
```
